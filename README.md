# PlayRewards - YouTube Reward Watch App

A responsive, high-end YouTube Reward tracking platform featuring advanced client watching state metrics verification with direct transactional RPC execution triggers on the database server.

## Features
- **Tab Focus Syncing**: Automatically pauses the tracking system if user switches browser views.
- **Transactional Database Processing**: Rewards are computed securely via RPC inside database engines, preventing API payload tampering.
- **Referral Tracking**: Integrated rewards trigger tracking upon completion of referred user tasks.

## Setup Instructions
1. Run the database setup schema inside the SQL Editor inside your Supabase project console.
2. Replace credentials inside `js/supabase.js`.
3. Launch `index.html` to begin.
